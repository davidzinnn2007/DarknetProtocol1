# DARKNET: PROTOCOL

Projeto Android desenvolvido em Java no Android Studio.

Uma interface cyberpunk inspirada em terminais hackers, sistemas futuristas e jogos de infiltração digital.

---

# Objetivo do Projeto

O aplicativo foi criado como projeto acadêmico para demonstrar:

* Navegação entre telas
* Interface personalizada
* Manipulação de Activities
* Uso de XML no Android
* Sistema de missões
* Terminal simulado
* Estrutura organizada de projeto
* Experiência visual diferenciada

---

# Tecnologias Utilizadas

* Java
* Android Studio
* XML
* Material Design
* Android SDK

---

# Como Executar

1. Abrir o projeto no Android Studio
2. Esperar o Gradle sincronizar
3. Conectar um dispositivo Android ou iniciar um emulador
4. Executar o projeto utilizando o botão Run
